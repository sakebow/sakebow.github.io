  3. .sorted(Comparator.comparing(Fruit::getPrice))：对筛选出的水果按照价格排序。

最后说说为什么要用stream

stream的优点在于，我们能够很方便的进行筛选与排序，并且能够很方便的进行map操作。

实际上，stream的效率往往是不如for循环的。不管你数据量是大还是不大，效率就是不如人家。

但是呢，stream写代码的过程中，能变得简洁（说白了就是能缩行），所以stream好！
